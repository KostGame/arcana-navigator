# Changelog

## [Unreleased]

- Заменены текстовые labels группы карты на компактные символы с `aria-label` и `title`.
- Уплотнены мобильные действия в позициях режима «Собрать расклад»: вместо широких кнопок используются компактные icon-кнопки.
- Добавлен блок «Суть по открытым картам» с общей фразой для чтения расклада.
- Карточки выбранных карт в собранном раскладе стали короче: смысл, глаголы и подробности по раскрытию.
- Добавлен процесс хранения задач и отчётов Codex в `docs/agent-tasks` и `docs/agent-reports`.
- Сделаны компактные phrase-first выводы для быстрого разбора и собранного расклада.
- Общий вывод расклада больше не склеивает длинные summary всех карт.
- Добавлены короткая линия расклада, главный акцент, фраза для чтения и отдельный совет.
- Полный смысл, внимание, анти-трактовка и сборка смысла спрятаны в закрытый блок «Подробнее».
- Сокращены служебные тексты в hero и режиме «Собрать расклад».
- Добавлены тесты компактного summary расклада.

## [0.4.0] - 2026-05-07

- Перенесён выбор карты в активную позицию режима «Собрать расклад».
- Убран отдельный нижний блок выбора карты для пошагового расклада.
- Добавлена кнопка «Выбрать эту карту» для закрепления текущей комбинации без изменения селектов.
- Добавлена кнопка «Отмена» для режима изменения карты без потери текущего выбора.
- Добавлена мягкая прокрутка к активной позиции после выбора, очистки или изменения.
- Добавлен тест отмены редактирования позиции.

## [0.3.0] - 2026-05-06

- Добавлен режим «Собрать расклад» рядом с быстрым разбором.
- Добавлена чистая модель `SpreadSession` для пошагового заполнения позиций.
- Выбор карты в активной позиции закрепляет её сразу и переводит фокус на следующую пустую позицию.
- Заполненная позиция защищена от случайной перезаписи; замена доступна через «Изменить».
- Добавлены промежуточный вывод и итог расклада на основе уже заполненных позиций.
- Добавлены тесты логики сессии расклада.

## [0.2.0] - 2026-05-06

- Разделены схемы раскладов и типы вопросов.
- Добавлены 10 схем раскладов и 12 типов вопросов.
- Быстрый режим теперь учитывает тип вопроса при сборке трактовки.
- Добавлены проверки для Луны и 10 Кубков в разных вопросах.

## [0.1.0] - 2026-05-06

- Добавлен первый рабочий быстрый режим чтения карты: расклад, позиция, карта, ориентация.
- Добавлены 4 типа раскладов: профессия, отношения, выбор, день.
- Добавлены компактные справочники по мастям, достоинствам, придворным картам и старшим арканам.
- Добавлена композиционная модель чтения карты с точечными overrides.
- Настроены CI и GitHub Pages workflow.
