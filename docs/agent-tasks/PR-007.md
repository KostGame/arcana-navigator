# PR-007: Replace card group labels with compact icons

## 1. Контекст

Проект: `KostGame/arcana-navigator`.

После merge PR-006 режим «Собрать расклад» стал компактнее, но живая проверка на телефоне показала новую UX-проблему: в inline picker кнопки группы карты всё ещё не помещаются комфортно в одну строку.

На мобильной ширине подписи `Младшие`, `Двор`, `Старшие` вместе с селектами занимают слишком много места, текст обрезается и визуально спорит с формой выбора карты.

Задача взята из issue #9: https://github.com/KostGame/arcana-navigator/issues/9

## 2. Цель

Сделать переключатель группы карты компактнее на мобильном экране, заменив текстовые подписи пиктограммами с доступными полными названиями.

## 3. Ветка и PR

- Ветка: `feature/pr007-card-group-icons`
- PR: `PR-007: Replace card group labels with compact icons`
- Статус PR: Draft

## 4. Что сделать

1. Заменить видимые текстовые кнопки группы карты:
   - `Младшие` -> `◆`;
   - `Двор` -> `♛`;
   - `Старшие` -> `★`.
2. Добавить понятные доступные подписи:
   - `aria-label="Младшие арканы"`;
   - `aria-label="Придворные карты"`;
   - `aria-label="Старшие арканы"`;
   - `title` с полным названием.
3. Сохранить визуально понятное активное состояние.
4. Уплотнить сетку picker так, чтобы на ширине 390px группа карты и селекты не конфликтовали.
5. Проверить сценарии:
   - младшая карта: группа, масть, достоинство, ориентация;
   - придворная карта: группа, масть, придворная карта, ориентация;
   - старший аркан: группа, старший аркан, ориентация.
6. Не сломать быстрый режим. Предпочтительно применить компактные подписи единообразно и там тоже.

## 5. Что не делать

- Не добавлять внешние иконки или библиотеки.
- Не добавлять изображения.
- Не делать большой редизайн.
- Не менять смысловые данные карт.
- Не добавлять backend.
- Не добавлять API.
- Не добавлять localStorage.
- Не добавлять генератор случайных карт.

## 6. Тесты и проверки

Запустить:

## 7. Ручная проверка

На ширине 390px проверить:

- кнопки группы карты помещаются в строку;
- активное состояние понятно;
- элементы не налезают друг на друга;
- нет горизонтальной прокрутки;
- выбор младших, придворных и старших карт работает;
- быстрый режим не сломан.
