# Agent Tasks

Здесь хранятся постановки задач для Codex.

- Каждый файл задачи называется `PR-XXX.md`.
- Задача должна содержать цель, контекст, ограничения, проверки и формат отчёта.
- После выполнения задачи Codex открывает PR и пишет отчёт в `docs/agent-reports/PR-XXX.md`.
- Если задача меняет UX, в ней должны быть ручные сценарии проверки.
