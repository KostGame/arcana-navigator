# Agent Tasks

Здесь хранятся постановки задач для Codex.

- Каждый файл задачи называется `PR-XXX.md`.
- Задача должна содержать цель, контекст, ограничения, проверки и формат отчёта.
- После выполнения задачи Codex открывает обычный PR без Draft-статуса и пишет отчёт в `docs/agent-reports/PR-XXX.md`.
- Если есть риск или непроверенный UI, Codex честно указывает это в отчёте, но не использует Draft по умолчанию.
- Если задача меняет UX, в ней должны быть ручные сценарии проверки.
