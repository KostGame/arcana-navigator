# PR-011: Add standalone card reference screen

## 1. Контекст

Сейчас справочная информация по картам есть внутри быстрых режимов и вспомогательных блоков, но пользователю нужен отдельный самостоятельный справочник карт.

Сценарий: пользователь не обязательно делает расклад прямо сейчас. Он хочет открыть отдельный экран, быстро найти карту и посмотреть её базовый смысл, короткую суть, глаголы и подробности.

Задача взята из issue #17: https://github.com/KostGame/arcana-navigator/issues/17

## 2. Цель

Добавить отдельный режим `Справочник карт`, который работает независимо от быстрого разбора и режима «Собрать расклад».

Главная задача: быстро найти карту и понять, как о ней говорить.

## 3. Ветка и PR

- Ветка: `feature/pr011-standalone-card-reference`
- PR: `PR-011: Add standalone card reference screen`
- Статус PR: Open / Ready for review

## 4. Что сделать

1. Добавить отдельную вкладку верхнего переключателя:
   - `Быстрый разбор`;
   - `Собрать расклад`;
   - `Справочник карт`.
2. Сделать отдельный экран `Справочник карт`.
3. Поддержать группы:
   - Старшие арканы;
   - Младшие арканы;
   - Придворные карты;
   - Масти;
   - Достоинства.
4. Добавить быстрые фильтры:
   - `Все`;
   - `Старшие`;
   - `Жезлы`;
   - `Кубки`;
   - `Мечи`;
   - `Пентакли`;
   - `Двор`.
5. Добавить локальный поиск по названию и смыслу: `Найти карту или смысл...`.
6. Сделать компактные карточки:
   - название;
   - короткая суть;
   - глаголы;
   - закрытый блок `Подробнее`.
7. В `Подробнее` показать полный смысл, плюс, минус, совет, внимание, анти-трактовку, глаголы и сборку значения.
8. Для мастей и достоинств добавить отдельные компактные блоки.
9. Использовать существующие data-файлы и `shortMeaning`; младшие и придворные карты собирать программно.

## 5. Что не делать

- Не добавлять backend.
- Не добавлять внешние API.
- Не добавлять генератор случайных карт.
- Не добавлять историю раскладов.
- Не добавлять localStorage.
- Не добавлять изображения карт.
- Не делать большой редизайн.
- Не переписывать все длинные значения карт.

## 6. Тесты и проверки

Минимум проверить:

1. Справочник может собрать все 22 старших аркана.
2. Справочник может собрать младшую карту `8 Кубков` с `shortMeaning`.
3. Справочник может собрать придворную карту `Паж Пентаклей` с `shortMeaning`.
4. Поиск или фильтр возвращает ожидаемые карты.
5. Быстрый режим и режим «Собрать расклад» не сломаны.

Запустить:

- `npm run check`
- `npm test`
- `npm run build`
- `git diff --check`

## 7. Ручная проверка

На ширине 390px проверить:

- появилась отдельная вкладка `Справочник карт`;
- можно быстро открыть список старших арканов;
- можно найти или отфильтровать `Луна`;
- можно найти или отфильтровать `8 Кубков`;
- можно найти или отфильтровать `Паж Пентаклей`;
- карточки компактные;
- подробности раскрываются только по клику;
- горизонтальной прокрутки нет;
- быстрый режим и «Собрать расклад» продолжают работать.
