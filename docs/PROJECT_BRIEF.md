# Project Brief

## Цель

Arcana Navigator помогает во время живого расклада быстро понять, как читать конкретную карту на конкретной позиции. Это справочный навигатор, а не приложение для гадания.

## Целевая аудитория

- Практикующие, которые работают с физической колодой.
- Новички, которым нужна мягкая опора для устного чтения.
- Люди, которые хотят говорить о карте спокойно, без категоричных предсказаний.

## Основные сценарии

- Выбрать расклад и позицию.
- Выбрать карту и ориентацию.
- Получить краткий смысл, глаголы и готовые фразы.
- Быстро открыть справочник по мастям, достоинствам, двору или старшим арканам.

## Ограничения

- Только статический сайт.
- Без backend, внешних API, авторизации и базы данных.
- Без генерации случайных карт и раскладов.
- Контент формулируется как гипотеза и опора для чтения, а не как приговор.
