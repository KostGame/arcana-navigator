# Checks

## Команды

```bash
npm install
npm run check
npm test
npm run build
git diff --check
```

## Локальная проверка

- Запустить `npm run dev`.
- Открыть локальный адрес Vite.
- Проверить быстрый режим: расклад → тип вопроса → позиция → карта → ориентация.
- Проверить режим «Собрать расклад»: выбор карты находится внутри активной позиции.
- Выбрать несколько карт подряд и убедиться, что заполненная позиция схлопывается, а следующая раскрывается.
- Проверить, что «Изменить» открывает picker внутри заполненной позиции.
- Проверить, что «Отмена» закрывает режим изменения без сброса карты.
- Проверить, что «Очистить» очищает одну позицию, а «Очистить расклад» очищает всё.
- Проверить, что смена типа вопроса сохраняет карты и пересчитывает трактовки.
- Проверить, что смена расклада с выбранными картами показывает подтверждение очистки.
- Проверить, что справочник раскрывается вертикально.

## Мобильная ширина

- Проверить ширину около 390px.
- Убедиться, что нет горизонтальной прокрутки.
- Проверить, что sticky-блок результата не перекрывает элементы.
- Проверить, что inline picker, селекты и кнопки «Изменить», «Очистить», «Отмена» не вылезают за экран.
