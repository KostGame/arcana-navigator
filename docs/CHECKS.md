# Checks

```bash
npm install
npm run check
npm test
npm run build
git diff --check
```

## Локальная проверка

- Запустить `npm run dev`.
- Открыть локальный адрес Vite.
- Проверить быстрый режим: расклад → тип вопроса → позиция → карта → ориентация.
- Проверить, что быстрый режим сначала показывает суть, глаголы и одну фразу.
- Проверить режим «Собрать расклад»: выбор карты находится внутри активной позиции.
- Выбрать 3-4 карты подряд и убедиться, что по каждой карте видно короткую суть без чтения длинного текста.
- Проверить, что подробности доступны через закрытый блок «Подробнее».
- Проверить, что общий вывод расклада короткий: заполнено, линия, акцент, фраза и совет.
- Проверить «Изменить», «Отмена», «Очистить» и «Очистить расклад».
- Проверить, что смена типа вопроса сохраняет карты и пересчитывает трактовки.
- Проверить, что смена расклада с выбранными картами показывает подтверждение очистки.
- Проверить «Справочник карт»: найти Луну, 8 Кубков и Пажа Пентаклей, добавить их через `+ В расклад` и перейти кнопкой `К раскладу`.
- Проверить добавление карты из справочника в конкретную выбранную позицию.
- Проверить предупреждение при попытке добавить карту из справочника в заполненную позицию.
- Проверить фильтры справочника: `★`, `◆`, `♛`, масти, `8`, `Паж`.
- Проверить, что заполненные позиции не перезаписываются при добавлении из справочника.
- Проверить sticky-строку карт в режиме «Собрать расклад»: тап по заполненной карте открывает замену.

## Мобильная ширина

- Проверить ширину около 390px.
- Убедиться, что нет горизонтальной прокрутки.
- Проверить, что inline picker, селекты и кнопки не вылезают за экран.
- Проверить, что блок результата не превращается в длинную простыню.
