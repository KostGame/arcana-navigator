# Roadmap

## PR-000: Bootstrap + первая рабочая версия

- Vite + TypeScript.
- Быстрый режим чтения карты.
- 3-4 типа раскладов.
- Базовый справочник карт.
- GitHub Actions и Pages workflow.

## PR-001: Расклад отдельно, вопрос отдельно

- Разделить типы раскладов и типы вопросов.
- Добавить 10 схем раскладов и 12 смысловых линз вопросов.
- Обновить быстрый режим: расклад → тип вопроса → позиция → карта → ориентация.
- Добавить первые overrides для Луны и 10 Кубков в разных типах вопросов.

## PR-002: Guided spread session mode

- Добавить режим «Собрать расклад».
- Закреплять выбранные карты по позициям без генерации случайных карт.
- Защитить заполненные позиции от случайной перезаписи.
- Пересчитывать трактовки и общий вывод после каждого выбранного места.
- Сохранить быстрый режим без изменений сценария.

## PR-003: Improve core meanings and spoken reading phrases

- Углубить значения всех старших арканов.
- Расширить масти, достоинства и придворные карты.
- Добавить больше живых фраз для устного чтения.
- Добавить больше точечных overrides для частых сочетаний.
- Улучшить трактовки перевёрнутых карт.

## PR-004: Мобильный UX, поиск и фильтры

- Добавить поиск по карте и смыслу.
- Улучшить мобильную навигацию по справочнику.
- Проверить типографику и плотность карточек.

## PR-005: PWA / offline mode

- Добавить manifest и service worker.
- Сделать сайт доступным без интернета после первого открытия.

## PR-006: Пользовательские расклады

- Добавить локальные пользовательские схемы.
- Сделать экспорт и импорт своих раскладов без сервера.
