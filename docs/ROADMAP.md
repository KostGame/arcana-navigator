# Roadmap

## PR-000: Bootstrap + первая рабочая версия

- Vite + TypeScript.
- Быстрый режим чтения карты.
- Базовые типы раскладов.
- Базовый справочник карт.
- GitHub Actions и Pages workflow.

## PR-001: Расклад отдельно, вопрос отдельно

- Разделить типы раскладов и типы вопросов.
- Добавить 10 схем раскладов и 12 смысловых линз вопросов.
- Обновить быстрый режим: расклад → тип вопроса → позиция → карта → ориентация.
- Добавить первые overrides для Луны и 10 Кубков.

## PR-002: Guided spread session mode

- Добавить режим «Собрать расклад».
- Закреплять выбранные карты по позициям без генерации случайных карт.
- Защитить заполненные позиции от случайной перезаписи.
- Пересчитывать трактовки и общий вывод после каждого выбранного места.

## PR-003: Improve guided spread card picker UX

- Перенести выбор карты внутрь активной позиции.
- Добавить явную кнопку «Выбрать».
- Добавить отмену редактирования.
- Уплотнить mobile picker.

## PR-004: Make readings compact and phrase-first

- Сделать быстрый и сессионный выводы компактными.
- Показывать сначала суть, глаголы и фразу.
- Спрятать подробности в «Подробнее».
- Сократить общий вывод расклада.

## PR-005: Agent task/report workflow

- Добавить `docs/agent-tasks` для постановок задач Codex.
- Добавить `docs/agent-reports` для отчётов Codex.
- Добавить шаблоны задачи и отчёта.
- Обновить правила будущих итераций в `AGENTS.md`.

## Следующие итерации

- Углубить значения карт и spoken reading phrases.
- Улучшить поиск и фильтры.
- Добавить PWA / offline mode.
- Исследовать пользовательские расклады без backend.
