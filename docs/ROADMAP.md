# Roadmap

## PR-000: Bootstrap + первая рабочая версия

- Vite + TypeScript.
- Быстрый режим чтения карты.
- 3–4 типа раскладов.
- Базовый справочник карт.
- GitHub Actions и Pages workflow.

## PR-001: Расширение данных

- Углубить значения всех старших арканов.
- Расширить масти, достоинства и придворные карты.
- Добавить больше точечных overrides для частых сочетаний.

## PR-002: Быстрый режим выбора карты и позиции

- Ускорить выбор карты.
- Добавить улучшенные фильтры и быстрые presets.
- Сделать сохранение последнего выбора в браузере.

## PR-003: Мобильный UX, поиск и фильтры

- Добавить поиск по карте и смыслу.
- Улучшить мобильную навигацию по справочнику.
- Проверить типографику и плотность карточек.

## PR-004: PWA / offline mode

- Добавить manifest и service worker.
- Сделать сайт доступным без интернета после первого открытия.

## PR-005: Пользовательские расклады

- Добавить локальные пользовательские схемы.
- Сделать экспорт и импорт своих раскладов без сервера.
