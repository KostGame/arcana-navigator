# Отчёт Codex по PR-012

## 1. Кратко

Добавлено действие `+ В расклад` в отдельном справочнике карт: старшие, младшие и придворные карты можно отправить в текущую сессию расклада без повторного выбора через picker. Заполненные позиции не перезаписываются, а при полном раскладе показывается мягкое сообщение.

## 3. Что реализовано

- В `CardReferenceEntry` добавлено поле `selection` для старших арканов, младших карт и придворных карт.
- В справочнике у конкретных карт появилась компактная кнопка `+ В расклад`.
- Карта из справочника добавляется в активную пустую позицию текущего расклада.
- Если активная позиция заполнена, карта добавляется в следующую пустую позицию.
- Если все позиции заполнены, карты не перезаписываются и показывается сообщение: `Все позиции заполнены. Очистите позицию или измените карту в раскладе.`
- После добавления показывается статус `Добавлено: карта → позиция` и кнопка `К раскладу`.
- Ориентация карты из справочника по умолчанию: прямая.
