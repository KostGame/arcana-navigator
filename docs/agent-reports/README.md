# Agent Reports

Здесь хранятся отчёты Codex по каждому PR.

- Каждый отчёт называется `PR-XXX.md`.
- Отчёт должен быть на русском языке.
- Отчёт должен содержать ссылки на PR, ветку, коммит, проверки, риски и готовность.
- В отчёте нужно честно указывать ошибки, ограничения и непроверенные сценарии.
- В чат пользователь может отправлять только короткую ссылку на файл отчёта.
