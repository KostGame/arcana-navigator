# Arcana Navigator

Русскоязычный статический сайт «Навигатор раскладов Таро» для чтения карты на конкретной позиции расклада.

Сайт помогает человеку, который уже делает расклад физической колодой: выбрать расклад, тип вопроса, позицию, карту и ориентацию. В интерфейсе сначала показываются короткая суть, глаголы и фраза для устного чтения; подробности можно раскрыть отдельно.

Важно: тип расклада и тип вопроса разделены. Расклад задаёт схему карт и позиции, а тип вопроса задаёт смысловую линзу чтения.

Это справочный навигатор. Он не генерирует расклады, не вытягивает случайные карты и не заменяет медицинскую, юридическую или финансовую консультацию.

## Режимы

- Быстрый разбор: одна карта читается через цепочку расклад → тип вопроса → позиция → карта → ориентация.
- Собрать расклад: карты закрепляются по позициям выбранной схемы, трактовки пересчитываются после каждого выбора, а sticky-строка выбранных карт помогает быстро открыть позицию на замену.
- Справочник карт: отдельный поиск по картам и смыслам; конкретную карту можно добавить в следующую или выбранную позицию текущего расклада.
