# Arcana Navigator

Русскоязычный статический сайт «Навигатор раскладов Таро» для чтения карты на конкретной позиции расклада.

Сайт помогает человеку, который уже делает расклад физической колодой: выбрать тип расклада, тип вопроса, позицию, карту и ориентацию, а затем быстро получить краткий смысл, глаголы, стартовые фразы, фокус внимания и анти-трактовку.

Важно: тип расклада и тип вопроса разделены. Расклад задаёт схему карт и позиции, а тип вопроса задаёт смысловую линзу чтения.

Это справочный навигатор. Он не генерирует расклады, не вытягивает случайные карты и не заменяет медицинскую, юридическую или финансовую консультацию.

## Режимы

- Быстрый разбор: одна карта читается через цепочку расклад → тип вопроса → позиция → карта → ориентация.
- Собрать расклад: карты закрепляются по позициям выбранной схемы, трактовки пересчитываются после каждого выбора, а общий блок показывает промежуточный вывод или итог расклада. Выбор карты открывается прямо внутри активной позиции.
